About HTML

HTML stands for HyperText Markup Language. It is used to structure web pages.

HTML Elements

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a href="url">Link Text</a>
  • Images: <img src="path" alt="description">
  • Lists: <ul> for unordered lists
  • Tables: <table>
  • Forms: <form> for collecting user input

Accessibility

Using alt text for images helps users with disabilities:

  • <img src="path" alt="A description of the image">

Embedding Media

Embed a video in your page using the <video> tag:

            <video controls>
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>
